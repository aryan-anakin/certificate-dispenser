# Project rules — certificate-dispenser

## This is NOT the Next.js you know

This project uses a **modified version of Next.js with breaking changes**. APIs,
conventions, and file structure may all differ from what's in training data. Do
not assume any Next.js behavior from memory.

**Rule: Before writing or modifying any Next.js code, read the relevant guide in
`node_modules/next/dist/docs/` first.** Heed all deprecation notices.

Docs are organized as:

- `node_modules/next/dist/docs/01-app/` — App Router
- `node_modules/next/dist/docs/02-pages/` — Pages Router
- `node_modules/next/dist/docs/03-architecture/` — architecture
- `node_modules/next/dist/docs/04-community/` — community
- `node_modules/next/dist/docs/index.md` — entry point

When unsure which guide applies, start at `index.md` and follow it to the
specific feature before touching code.
